**产出：**
- [ ] DB migration：bids 表 + orders 表

- [ ] 竞拍状态机：pending → active → ended_sold/ended_no_bid/cancelled

## 五、风险 & 应对

| 风险 | 概率 | 影响 | 应对策略 |
|------|------|------|---------|
| 并发 Bug（余额不一致） | 高 | 致命 | Redis 锁 + DB 乐观锁双保险，写充分测试 |
| WebSocket 断连丢消息 | 中 | 高 | 心跳 + 断线补发状态快照 |
| 工期不够 | 中 | 高 | P1 功能可简化：看板用表格代替图表，动画用 CSS 基础版 |
| Go 并发模式不熟 | 低 | 中 | goroutine + channel + sync.Mutex，AI 辅助 |
| 前端动画复杂度高 | 中 | 中 | 先做 CSS 动画，有时间再加 Lottie |
